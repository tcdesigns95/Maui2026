🔧 Firebase Setup (one-time)

To sync in real time across all 5 travelers, connect a free Firebase Realtime Database:

  1. Go to console.firebase.google.com
  2. Add project → name it maui-wowie-2026
  3. Build → Realtime Database → Create → Start in test mode
  4. Project Settings ⚙️ → Your apps → Add web app → copy firebaseConfig
  5. Paste config values into FIREBASE_CONFIG in the script section
